# Todo PWA 开发计划

使用纯 HTML/CSS/JavaScript 构建待办事项应用，已逐步升级为 PWA。

---

## 第一阶段：基础 Todo 应用 ✅

- 添加 / 完成 / 删除任务
- 任务统计、空状态、清空已完成

## 第二阶段：本地持久化 ✅

- localStorage 存取，刷新不丢失
- 任务模型：`id`、`text`、`done`、`myDay`、`createdAt`

## 第三阶段：PWA 升级 ✅

- manifest.json（独立窗口、主题色）
- service-worker.js（缓存优先、离线可用）
- icon.svg
- 在线/离线状态浮动横幅

## 第四阶段：增强体验 ✅

- 搜索框（实时过滤）
- 过滤器（全部 / 未完成 / 已完成）
- 原地编辑（双击修改、Enter 确认、blur 取消）
- "我的一天"星标功能
- 创建时间显示

---

## 下一步可选方向

| 序号 | 功能 | 说明 |
|------|------|------|
| A | **拖拽排序** | 拖动任务项调整顺序，同步到 localStorage |
| B | **截止日期** | 任务加截止时间，过期高亮提醒 |
| C | **分类/标签** | 自定义标签，按标签筛选 |
| D | **深色模式** | 跟随系统 / 手动切换 |
| E | **数据导出/导入** | JSON 导出备份，导入恢复 |
| F | **键盘快捷键** | Ctrl+K 搜索、Ctrl+N 新建等 |
| G | **通知提醒** | Notification API，截止任务推送 |
| H | **热力图/统计** | 完成率日历热力图 |

---

你选哪个方向？也可以组合，或者提自己的想法。
