# Todo PWA 开发计划

## 项目概述

使用纯 HTML/CSS/JavaScript 创建一个 Todo 待办事项应用，并逐步升级为可离线使用的 PWA。

---

## 第一阶段：基础 Todo 应用 ✅

> 已完成

### 项目结构

```
/to do/
  index.html          # 主页面，包含完整 HTML 结构
  css/
    style.css         # 样式文件
  js/
    app.js            # 核心逻辑
  PLAN.md             # 本规划文档
```

### 功能

1. **添加任务** — 顶部输入框 + 添加按钮 / Enter 键
2. **完成任务** — 左侧复选框勾选，文字加删除线
3. **删除任务** — 悬停显示 × 按钮
4. **任务统计** — 底部显示 "已完成/总数"
5. **空状态** — 无任务时显示提示
6. **清空已完成** — 有已完成任务时显示清空按钮
